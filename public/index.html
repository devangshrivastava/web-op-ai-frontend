Welcome to AI Playground

Overview

AI Playground is a static, multi‐page environment designed to let both humans and AI agents seamlessly interact with common UI components. You’ll find search bars, filters, dynamic lists, settings panels, data tables, and more—all wired to console‐log their state changes so you can verify AI‐driven workflows in real time.

Every element has a unique ID and clear action contract, making it ideal for function-calling with OpenAI and scripted Playwright runs.
